## KMP 概述
:::danger
若本次講解與老師授課內容不同，則以老師為主
:::
- Knuth-Morris-Pratt
- 字串搜尋演算法
- `KMP` V.S `strstr`?
    - `KMP`：一次可回傳多個相同字串的位置
    - `strstr`：一次只會回傳第一個遇到匹配字串的首相位置

### 舉個🌰

```:
String T="ABABABACAC"//目標、被搜尋對象
String S="ABABACAC"//匹配字串
```

### Failure Function

1. 先找出Failure Function
2. 把握以下原則：
    - 從頭編號為-1，找到與第一個重複結束（前綴）
    - 與第一個字母重複則編為0，之後相同加1，後續依此類推
    - 當遇到非前綴字母且非首個字母時，編為-1

|比較的字串(S)| A | B | A | B | A | C | A | C |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |
|陣列位置(Pos)|0|1|2|3|4|5|6|7|
|Failure Function(FF)|-1|-1|0|1|2|-1|0|-1|

### 匹配

- 匹配的原則
1. 若 $S[Pos].TT<0$,則以他為始，往左找最近的$S[P].TT=0$的位置，把預匹配開頭放到與齊位再往下匹配，找不到0則直接往下一項開始匹配
2. 若 $S[Pos].TT>=0$,則直接往下一項開始匹配
