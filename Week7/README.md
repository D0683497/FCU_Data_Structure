# 資料結構實習[Week7-堆疊與佇列]

## 堆疊 Stack
- ==後進先出 LIFO== Last In First Out 。
- 最早放進去的東西會最後才會被取出來，最後放進去的東西則會是第一個拿出來的。
![](https://i.imgur.com/pAlHz93.jpg)
- 可利用陣列或鏈結串列實作。
- 以陣列為例:
    - ```create()```: 建立一個容量為MAX的空堆疊。
        - 將陣列內容初始化為0。
        - **將top設為-1**。
    - ```isFull()```: 堆疊是否已滿。
    - ```isEmpty()```:堆疊是否為空。
    - ```push()```: 將item放在stack**頂端**。
        - **若堆疊已滿則顯示堆疊已滿，且不增加**。
    - ``` pop()``` : 移除並**回傳堆疊頂端的元素**。
        - **若堆疊為空則顯示堆疊已空，且不移除**。
## 佇列 Queue
- ==先進先出 FIFO== First In First Out 。
- 最早放進去的東西會最先被取出來。
![](https://i.imgur.com/RVO8oA6.jpg)
- 可利用陣列或鏈結串列實作。
- 新增資料：必須從**後端**(**rear**)進行新增，不能從其他位置新增進去。
- 刪除資料：必須從**前端**(**front**)進行刪除，不能從其他位置刪除。
- 以陣列為例:
    ![](https://i.imgur.com/QZ77x2t.jpg)
    - ```create()```: 建立一個容量為MAX的空佇列。
        - 將陣列內容初始化為0。
        - **將front、rear設為0**。
    - ```isFull()```: 佇列是否已滿。
        - ==為了分辨環狀佇列是**已空**或**全滿**，其實際的儲存空間是**MAX-1**==。
        - **(rear+1)%MAX是否等於front**。
    - ```isEmpty()```: 佇列是否為空。
        - **front是否等於rear**。
    - ```add()```: 將item放在rear的位置。
        - **若佇列已滿則顯示佇列已滿，且不增加**。
    - ```del()```: 移除並回傳佇列最前端的元素。
        - **若佇列為空則顯示佇列已空，且不移除**。

### 3.中置轉後置實作
> 利用上述堆疊funtion實作中置轉後置。
- ==遇到 (== ：加入至堆疊。
- ==遇到運算子==：比較堆疊內頂端之運算子優先權，若堆疊內頂端之運算子優先權較大則輸出，最後將該運算子加入堆疊。
    - 運算子： (、) 、 + 、 - 、 * 、 / 、 % 。
    - 運算元：單一數字之整數或一個字元之變數 。
    - 優先權順序(參考正課簡報P.37)
- ==遇到 ) 輸出至 (==。
- ==若為運算元則直接輸出==。
- 最後將堆疊內所有運算元輸出。

輸入
```
(1+3)*5-2
```
輸出
```
13+5*2-
```
